pplogでは、いろんなことができません。

  • つながれない

    ともだちや好きな人のポエムを購読することができます。でも誰に読まれているかはポエムの持ち主にはわからないので、気軽に好きな人のポエムだけ読めます。

  • のこせない

    自分のポエムページには最新の一件だけが出ます。過去のポエムは、自分しか読むことができません。ポエムごとのURLもありません。

  • しゃべれない

    コメント欄もありません。 ボタンであしあとをつけることができます。 は文章を選択してつけることができ、そこだけあとから読むことができます。
